# 週次幸福メトリクス

個人の週次指標を 3 分で入力し、8 指標の合計スコアとレッドフラグを即座に可視化するローカル専用の Web アプリです。データはブラウザの `localStorage` に保存され、JSON 形式でエクスポート / インポートできます。

## 主な特徴

- 🧭 直近週の判定を「🟢A / 🟡B / 🔴C」で表示
- 📈 8 指標 (深い仕事、創作、対面、睡眠、飲酒ルール逸脱、歩数、緊急資金、パイプライン) のスコアリング
- 🚩 睡眠・対面・無収入の自動レッドフラグと手動フラグに対応
- 📊 週次スコア推移を折れ線グラフで表示
- 📋 前週コピー / JSON エクスポート・インポート / 全データ削除

## 開発メモ

1. 依存関係をインストール

   ```bash
   npm install
   ```

2. 開発サーバーを起動

   ```bash
   npm run dev
   ```

3. 本番ビルド

   ```bash
   npm run build
   ```

> **Note:** 実行環境によってはパッケージレジストリへのアクセスが制限される場合があります。その場合はローカルにキャッシュされたモジュールや社内レジストリを利用してください。
